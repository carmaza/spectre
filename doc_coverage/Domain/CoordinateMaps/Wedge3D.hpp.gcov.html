      23             :  *
      24             :  * \details The mapping that goes from a reference cube to a three-dimensional
      25             :  *  wedge centered on a coordinate axis covering a volume between an inner
      26             :  *  surface and outer surface. One of the surfaces must be spherical, but the
      27             :  *  curvature of the other surface can be anything between flat (a sphericity of
      28             :  *  0) and spherical (a sphericity of 1).
      29             :  *
      30             :  *  The first two logical coordinates correspond to the two angular coordinates,
      31             :  *  and the third to the radial coordinate.
      32             :  *
      33             :  *  The Wedge3D map is constructed by linearly interpolating between a bulged
      34             :  *  face of radius `radius_of_other_surface` to a spherical face of
      35             :  *  radius `radius_of_spherical_surface`, where the radius of the bulged face
      36             :  *  is defined to be the radius of the sphere circumscribing the bulge.
      37             :  *
      38             :  *  We make a choice here as to whether we wish to use the logical coordinates
      39             :  *  parameterizing these surface as they are, in which case we have the
      40             :  *  equidistant choice of coordinates, or whether to apply a tangent map to them
      41             :  *  which leads us to the equiangular choice of coordinates. In terms of the
      42             :  *  logical coordinates, the equiangular coordinates are:
      43             :  *
      44             :  *  \f[\textrm{equiangular xi} : \Xi(\xi) = \textrm{tan}(\xi\pi/4)\f]
      45             :  *
      46             :  *  \f[\textrm{equiangular eta}  : \mathrm{H}(\eta) = \textrm{tan}(\eta\pi/4)\f]
      47             :  *
      48             :  *  With derivatives:
      49             :  *
      50             :  *  \f[\Xi'(\xi) = \frac{\pi}{4}(1+\Xi^2)\f]
      51             :  *
      52             :  *  \f[\mathrm{H}'(\eta) = \frac{\pi}{4}(1+\mathrm{H}^2)\f]
      53             :  *
      54             :  *  The equidistant coordinates are:
      55             :  *

      57             :  *
      58             :  *  \f[ \textrm{equidistant eta}  : \mathrm{H} = \eta\f]
      59             :  *

      61             :  *
      62             :  *  <center>\f$\Xi'(\xi) = 1\f$, and \f$\mathrm{H}'(\eta) = 1\f$</center>
      63             :  *
      64             :  *  We also define the variable \f$\rho\f$, given by:
      65             :  *
      66             :  *  \f[\textrm{rho} : \rho = \sqrt{1+\Xi^2+\mathrm{H}^2}\f]
      67             :  *

      69             :  *  The surface map for the spherical face of radius \f$R\f$ lying in the
      70             :  * \f$+z\f$
      71             :  *  direction in either choice of coordinates is then given by:
      72             :  *
      73             :  *  \f[\sigma_{spherical}: \vec{\xi} \rightarrow \vec{x}(\vec{\xi})\f]
      74             :  *  Where
      75             :  *  \f[
      76             :  *  \vec{x}(\xi,\eta) =
      77             :  *  \begin{bmatrix}
      78             :  *  x(\xi,\eta)\\
      79             :  *  y(\xi,\eta)\\
      80             :  *  z(\xi,\eta)\\
      81             :  *  \end{bmatrix}  = R
      82             :  *  \begin{bmatrix}
      83             :  *  \Xi/\rho\\
      84             :  *  \mathrm{H}/\rho\\
      85             :  *  1/\rho\\
      86             :  *  \end{bmatrix}\f]
      87             :  *
      88             :  *  ### The Bulged Face Map
      89             :  *  The bulged surface is itself constructed by linearly interpolating between
      90             :  *  a cubical face and a spherical face. The surface map for the cubical face
      91             :  *  of side length \f$2L\f$ lying in the \f$+z\f$ direction is given by:
      92             :  *

      94             :  *  Where
      95             :  *  \f[

     101             :  *  \end{bmatrix}  = L

     106             :  *  \end{bmatrix}\f]

     126             :  *  interpolating between the spherical and bulged surfaces with the

     139             :  *  second *blending factors*:

     141             :  *  \f[b_{f1} = \frac{1}{2}\big\{(1-\zeta)(1-s)L\big\}\f]
     142             :  *
     143             :  *  \f[b_{f2} = \frac{1}{2}\big\{(1-\zeta)sR_{other} + (1+\zeta)R_{spherical}
     144             :  *  \big\}\f]
     145             :  *
     146             :  *  We also define their zeta-derivatives, the *blending rates*:
     147             :  *
     148             :  *  \f[b_{r1} = \frac{\mathrm{d}b_{f1}}{\mathrm{d}\zeta} =
     149             :  *  \frac{1}{2}(s-1)L\f]

     162             :  *  b_{r2}\Xi/\rho\\

     166             :  *  \end{bmatrix}

     169             :  *  \note This differs from the choice in SpEC where it is demanded that the
